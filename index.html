Are the keys generated with this website secure?

Yes, same security as a software client. The key generation on this website is done client side only. The keys generated on this website never leave your browser.

Is it safe to use my private key on this website?

It should be safe. However I do not recommend using your main private key since there might be a flaw with this website, also you should never share your private key anywhere.

Why does my web browser or computer slow down when I'm generating keys?

Generating a PGP key takes resources so you will notice an increase of RAM and CPU usage. The better your hardware, the faster it will generate.
Pretty Good Privacy (PGP) is a data encryption and decryption computer program that provides cryptographic privacy and authentication for data communication. PGP is often used for signing, encrypting, and decrypting texts, e-mails, files, directories, and whole disk partitions and to increase the security of e-mail communications. It was created by Phil Zimmermann in 1991. PGP and similar software follow the OpenPGP standard (RFC 4880) for encrypting and decrypting data. Source: Wikipedia

This site provides a simple, easy to understand interface to generate PGP keys. I wanted to provide an easier way to generate keys since most clients are CLI or meant for advanced users None of this would be possible without the awesome Open Source code this website uses.
